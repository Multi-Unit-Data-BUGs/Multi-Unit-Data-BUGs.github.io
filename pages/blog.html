PCB for the Mothership â€“ MPCB v1.0

Posted on January 30th, 2025 by Burt Kim

Burt Kim has initiated the design of MPCB v1.0, a custom PCB for the Mothership, integrating ESP32, an RFM9x LoRa chip, and a microSD card holder. This redesign aims to enhance affordability and efficiency compared to the current Mothership architecture, which relies on ESP32 or Arduino Uno, an RFM9x chip, and a Raspberry Pi.

The new PCB will streamline data collection and storage by leveraging LoRaWAN communication. Incoming data from the RFM9x chip will be processed by the ESP32, and instead of relying on a Raspberry Pi for storage, the processed data will be stored directly on a microSD card. Additionally, the Mothership will continue uploading data to Firebase, ensuring cloud accessibility and real-time monitoring. By consolidating components and optimizing the data pipeline, MPCB v1.0 aims to create a cost-effective and scalable solution for efficient long-range data communication.

App Updates: Data Viewing and Mapping Functionality

Posted on September 4, 2024 by Burt Kim

We're thrilled to share the latest updates on our application! We've added two key features that enhance usability and functionality: a data viewing screen and a mapping functionality to log and view the placement of each module in the garden.

The data viewing screen allows users to select the device number, date, and soil metric to view specific data points. This makes it easier to track and analyze soil health over time. Additionally, the mapping feature provides a visual interface for manually logging and viewing the exact placement of each module in the garden.
